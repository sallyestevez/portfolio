Ktown4u Web Redesign

I led a comprehensive overhaul of the Ktown4u website to enhance the user experience, visual appeal, and functionality. By conducting user research and usability testing, I identified key pain points and gathered insights to inform the redesign. I developed wireframes, prototypes, and high-fidelity mockups using Figma, focusing on improving navigation and information architecture.

Retail Horror Stories: Black Friday Edition

I created a "Choose Your Own Adventure" game inspired by real retail workers' experiences during Black Friday. The player takes the role of a Walmart employee navigating different retail scenarios and making choices that impact the storyline. The game was made with HTML, CSS, and JavaScript. I also added background music to enhance the gaming experience.

Kirby's Corner

This is an interactive fanpage dedicated to the Kirby universe. The site features detailed information about games, characters, and lore, presented through a visually appealing and user-friendly interface. The website was created using HTML, CSS, and JavaScript. To add on to the site's interactivity, I included a music player and a quiz. The project was well-received at the IDM Senior Showcase thanks to its thorough research and cohesive design.

Eclipse Soundscapes

For this project, my group and I focused on enhancing the accessibility of a field researcher training page for older audiences. We gathered insights to improve usability by conducting user interviews with individuals aged 55 and older. We optimized navigation, typography, and contrast ratios to create a user-friendly interface that complies with WCAG guidelines. We created wireframes and used them to build a prototype in Figma.

Completed: Fall 2022

Miiverse Revival

This is a "revival" of the now-defunct Nintendo social media site called Miiverse. I wanted the user experience to feel as close to the original Miiverse as possible, so I replicated Miiverse's aesthetic. Users can make their own accounts and create posts that are then displayed on the Activity Feed. The website was created using HTML, CSS, JavaScript, and React.

Discord File Compressor

Discord does not allow users to send files larger than 8 MB. I proposed a Discord update that would include an in-app compression feature for larger files. I conducted user interviews to determine what Discord users were interested in. I also researched competitors to see how their features differ from Discord's. I created wireframes and used them to build a prototype in Figma.

Completed: Fall 2022
