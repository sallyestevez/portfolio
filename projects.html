Sun and Moon Information + Song of the Hour

The page returns sun and moon data based on the day. Users can switch between viewing sun and moon data. The page also returns a song based on the hour. The project was made using HTML, CSS, and React.js. The sun and moon data were drawn from an astronomy API. The song data was drawn from a JSON file created by me.

Completed: Fall 2022

Discord File Compressor

Discord does not allow users to send files larger than 8 MB. I proposed a Discord update that would include an in-app compression feature for larger files. I conducted user interviews to determine what Discord users were interested in. I also researched competitors to see how their features differ from Discord's. I creaated wireframes and used them to build a prototype in Figma.

Completed: Fall 2022
